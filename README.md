 # Movie Search - Single Page Application (SPA)

# Project Overview
This project is a "Single Page Application (SPA)" for searching and managing movie details.  
It allows users to search movies dynamically while an "Admin" can upload movie trailers, posters, keywords, and other movie details.
The application updates content **without refreshing the page** using AJAX.

# Features
- Admin login page
- Upload movie trailers
- Upload movie posters
- Add movie keywords and details
- Movie search functionality
- Dynamic movie list updates using AJAX
- Responsive user interface using Bootstrap
- 
# Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery
- AJAX
- 
# How It Works
1. Admin logs into the system.
2. Admin uploads movie trailer, poster, keywords, and details.
3. The form is submitted using "AJAX".
4. The movie list updates dynamically without refreshing the page.
